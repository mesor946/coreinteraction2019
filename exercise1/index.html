Exercise 1- Project Page




The purpose of this extension is to create a version of the immune system which would protect or defend the outside of the body. The artists explored the roles of the immune system and created visual representations of how an external immune system would appear.

Sea Urchin Rave Outfit, by Rebecca Mesonjnik, was created using mixed media, with the two main components being items of revealing clothing and cardboard cones. The outfit is directly confrontational, forcing the viewer to keep their distance, and defending the wearer from unwanted physical contact.
